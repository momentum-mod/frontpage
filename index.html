What is Momentum Mod?

Momentum Mod is a free, standalone game built on the Strata Source Engine that aims to faithfully recreate, centralize, and improve upon the movement game modes found in Counter-Strike, Team Fortress, Half-Life 2, Titanfall 2, and Quake.
